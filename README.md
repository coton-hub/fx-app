# Forex App — Starter

Structure minimale pour démarrer une application front-end (HTML/CSS/JS) avec Bootstrap et jQuery.

Fichiers clés:
- `index.html` — page d'entrée (inclut Bootstrap & jQuery via CDN)
- `css/main.css` — styles de base
- `js/main.js` — script de démarrage

Pour lancer rapidement un server local (recommandé) :

```bash
# depuis le dossier `forex app`
python3 -m http.server 8000
# puis ouvrir http://localhost:8000 dans votre navigateur
```

Optionnel: initialiser un dépôt Git :

```bash
cd "forex app"
git init
git add .
git commit -m "Initial project scaffold"
```
